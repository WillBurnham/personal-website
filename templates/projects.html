Real Estate Data Collector

This is a web scraping application that accesses Trulia.com for real estate data in the city and state the user chooses. It takes the values submitted through the form, searches Trulia for property data in the given city/state, and then compiles it into a spreadsheet.

Weather Reminder App

This project uses the Open Weather Map API. It fetches the next day's weather data and searches for trigger words that would indicate bad weather. When bad weather is detected, it sends me an email notifying me of bad weather in either Austin, Dallas, Houston, or any combination of the three. On the front end, the images below change dynamically dependent on the current state of the weather.
